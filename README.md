# GridLocator
Determine maidenhead grid location or lat/lon coordinates


    $ python3 grid.py --city "Gobblers Knob, PA"
    Coordinates and grid location for Gobblers Knob, PA:
    40.0848074, -77.9419441
    FN10ac


    $ python3 grid.py --grid DM46
    Center of grid DM46:
    (36.5, -111.0)
